### Requirements

- OpenCV + OpenCV Contrib

- ZED SDK: https://www.stereolabs.com/developers/

- YOLO model (model.onnx)

- Make sure your camera is properly connected and calibrated

### Build & Run

       git clone https://github.com/ichsanyudika12/crosswalk-monitoring.git
       
       cd crosswalk-monitoring
       
       mkdir build && cd build
       
       cmake ..
       
       make

       ./realtime
